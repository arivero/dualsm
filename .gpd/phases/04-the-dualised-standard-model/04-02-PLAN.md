  <action>
    Create scripts/verify_sm_anomaly_matching.py following the structure and style of the existing verify_anomaly_matching.py (VALD-02).

    The script must:

    1. **Define field content symbolically.** Use SymPy symbols for n_g (number of generations). Define all magnetic fields from Table II of 2407.17281 with their SU(3)_c, SU(2)_L, U(1)_Y quantum numbers. Define all electric fields from Table IV of 1102.5100 with their quantum numbers under the global symmetries.

    For the magnetic theory, the FERMIONS contributing to anomalies are:
    - SM quarks: q_L (3, 2, 1/6) x n_g, u_R (3, 1, 2/3) x n_g, d_R (3, 1, -1/3) x n_g (Weyl fermions)
    - SM leptons: l_L (1, 2, -1/2) x n_g, e_R (1, 1, -1) x n_g (Weyl fermions)
    - Gaugino: lambda_m in adjoint of SU(3)_c = (8, 1, 0) (one Weyl fermion)
    - Mesino: M in some representation (check Table II for exact quantum numbers)
    - Note: squarks, mes-Higgs are SCALARS -- they do NOT contribute to anomalies
    - The 18 Higgs doublets are SCALARS -- they do NOT contribute to anomalies
    - BUT: there may be Higgsino-like fermion partners in the quasi-SUSY spectrum -- check Table II carefully

    For the electric theory:
    - Electric quarks p, p-tilde in fundamental/antifundamental of SU(2) with SU(n_g)_L x SU(n_g)_R x SU(2)_L x SU(2)_R quantum numbers
    - Electric gaugino lambda in adjoint of SU(2)

    2. **Compute all 6 anomaly conditions.** For each condition, compute the total anomaly coefficient on both the electric and magnetic sides as a polynomial in n_g:
    - SU(3)_c^3: sum A(R_3) over colour-charged Weyl fermions
    - SU(2)_L^3: trivially zero (d_{abc} = 0 for SU(2))
    - SU(3)_c^2 U(1)_Y: sum Y * T(R_3) over colour-charged Weyl fermions
    - SU(2)_L^2 U(1)_Y: sum Y * T(R_2) over SU(2)_L doublet Weyl fermions
    - U(1)_Y^3: sum Y^3 over all Weyl fermions
    - Grav U(1)_Y: sum Y over all Weyl fermions

    3. **Symbolic verification.** Use sympy.simplify to verify electric - magnetic = 0 for each condition as a polynomial identity in n_g.

    4. **Numerical spot-check.** Substitute n_g = 3 and verify all conditions give integer zero.

    5. **SM subset check.** Remove extra magnetic states (gaugino, mesino) from the magnetic anomaly and verify that the remaining SM fields satisfy standard anomaly cancellation (all coefficients vanish independently per generation).

    6. **Output.** Print PASS/FAIL for each check with descriptive labels. Exit with return code 0 if all pass, 1 if any fail.

    Include ASSERT_CONVENTION line at the top of the file matching the project conventions.

    IMPORTANT: The field content must be read from 2407.17281 Tables I-II and 1102.5100 Tables III-IV. If the Lecture 7 field content table disagrees with these references, use the references as ground truth and flag the discrepancy.
