    Problem 2: Fayet-Iliopoulos D-term and SUSY Breaking (~25 min student time)
    - Consider a U(1) gauge theory with one chiral superfield Phi of charge q = +1.
    - Part (a): Write the D-term potential: V_D = (1/2)(g |phi|^2 + xi)^2. Find the minimum. For xi > 0: <|phi|^2> = -xi/g < 0, which is impossible, so SUSY is broken (V_D > 0). For xi < 0: <|phi|^2> = |xi|/g > 0, SUSY is preserved, U(1) is broken.
    - Part (b): For the SUSY-preserving case (xi < 0), compute the gauge boson mass: m_A^2 = 2g^2 |<phi>|^2 = 2g |xi|. Verify [m_A] = [mass] given [g] = [dimensionless] and [xi] = [mass]^2.
    - Part (c): Add a second chiral superfield Phi-tilde of charge q = -1 and superpotential W = m Phi Phi-tilde. Write the full scalar potential V = V_F + V_D. Find the vacuum for xi < 0 and m != 0. Show that both SUSY conditions (F = 0 and D = 0) can be satisfied simultaneously when xi < 0.
    - Part (d): Discuss: in the N=2 context, FI terms arise on the Coulomb branch for the U(1)^{N_c-1} factors. When monopoles become massless at singular Coulomb branch points, a nonzero FI parameter can trigger monopole condensation (analogous to part (a) with xi < 0). This is the mechanism for confinement in the dual picture.
    - Solution: full working for parts (a)-(c), conceptual discussion for (d).
