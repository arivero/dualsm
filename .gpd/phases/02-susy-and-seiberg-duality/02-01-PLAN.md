    STRUCTURE (following RESEARCH.md Section "Lecture 2: Minimal N=1 SUSY Introduction"):

    Section 1: Motivation (~5 min content)
    - Why SUSY? NOT phenomenological motivation (no superpartners observed). Instead: theoretical power -- holomorphy, exact results, non-renormalisation, electric-magnetic duality.
    - State the goal: SUSY provides the ONLY known framework where electric-magnetic duality can be PROVED (via anomaly matching). Understanding SUSY duality is a prerequisite for the conjectural non-SUSY extension.

    Section 2: The SUSY Algebra (~5 min)
    - State (do NOT derive) the N=1 SUSY algebra: {Q_alpha, Q-bar_{alpha-dot}} = 2 sigma^mu_{alpha alpha-dot} P_mu.
    - Note: Q is fermionic (spin-1/2 generator), so SUSY relates bosons and fermions.
    - State: each SUSY multiplet has equal numbers of bosonic and fermionic degrees of freedom.
    - Weyl spinor notation: alpha = 1,2 (undotted), alpha-dot = 1,2 (dotted). Briefly state sigma^mu matrices.

    Section 3: Chiral Superfields (~10 min)
    - Define superspace coordinates (x^mu, theta^alpha, theta-bar^{alpha-dot}).
    - Define chiral superfield: Phi(y, theta) = phi(y) + sqrt(2) theta psi(y) + theta^2 F(y) where y^mu = x^mu + i theta sigma^mu theta-bar.
    - Component fields: complex scalar phi, Weyl fermion psi, auxiliary field F.
    - DOF counting: 2 bosonic (complex scalar, off-shell) + 2 fermionic (Weyl fermion). After EOM: F is eliminated, 2B = 2F on-shell.
    - Superpotential: W(Phi) is a HOLOMORPHIC function of chiral superfields.
    - Lagrangian from superpotential: L_W = int d^2 theta W(Phi) + h.c. = F_i partial W/partial phi_i + ... (state the F-term extraction).

    Section 4: Vector Superfields and Gauge Theories (~10 min)
    - Define vector superfield V = V^dag (real superfield).
    - Wess-Zumino gauge: V = -theta sigma^mu theta-bar A_mu + i theta^2 theta-bar lambda-bar - i theta-bar^2 theta lambda + (1/2) theta^2 theta-bar^2 D.
    - Component fields: gauge boson A_mu, gaugino lambda (Weyl fermion in adjoint), auxiliary D field.
    - DOF counting: 3B (A_mu in WZ gauge) + 2F (gaugino) off-shell, or 2B + 2F on-shell after D elimination.
    - Field-strength superfield: W_alpha = -(1/4) D-bar^2 D_alpha V (chiral!).
    - Gauge kinetic term: L_gauge = (1/4g^2) int d^2 theta W^alpha W_alpha + h.c.
    - Gauge coupling of matter: Phi^dag e^{2gV} Phi in the Kahler potential.

    Section 5: The N=1 Lagrangian and Scalar Potential (~5 min)
    - General N=1 Lagrangian: L = int d^4 theta K(Phi^dag, e^{2gV} Phi) + int d^2 theta [W(Phi) + tau/(16 pi i) W^alpha W_alpha] + h.c.
    - For renormalisable theories: K = Phi^dag e^{2gV} Phi (canonical Kahler).
    - F-term potential: V_F = sum_i |partial W / partial phi_i|^2.
    - D-term potential: V_D = (1/2) sum_a (g sum_i phi_i^dag T^a phi_i)^2.
    - Total scalar potential: V = V_F + V_D >= 0. SUSY vacuum: V = 0, i.e., F_i = 0 AND D^a = 0.
    - Note: V >= 0 is a special feature of SUSY. If a SUSY vacuum exists, it is the true vacuum (no tunnelling needed).

    Section 6: The Non-Renormalisation Theorem (~5 min)
    - State the theorem: The superpotential W(Phi) receives NO perturbative corrections (to all orders in perturbation theory).
    - Present Seiberg's symmetry argument (1993): promote couplings to background superfields; holomorphy + symmetries + weak-coupling limits uniquely fix W.
    - Key consequence: the tree-level superpotential is EXACT perturbatively. Non-perturbative corrections (instantons) CAN modify W, but are exponentially suppressed.
    - State the NSVZ exact beta function: beta(g) = -(g^3/16pi^2)[3T(G) - sum_i T(R_i)(1 - gamma_i)] / [1 - T(G) g^2/(8pi^2)].
    - Verify: with T(fund) = 1/2 and T(adj) = N_c, this gives b_0 = (11N_c - 2N_f)/3 at one loop (gamma_i = 0), matching Phase 1.

    Section 7: N=1 SQCD -- Field Content and Symmetries (~5 min)
    - Define N=1 SQCD: SU(N_c) gauge theory with N_f chiral superfields Q^i in fundamental and N_f chiral superfields Q-tilde_i in antifundamental.
    - W_electric = 0 (no tree-level superpotential for SQCD).
    - Classical global symmetry: SU(N_f)_L x SU(N_f)_R x U(1)_B x U(1)_A x U(1)_R.
    - Anomalous symmetry: U(1)_A is broken by the ABJ anomaly (just like the non-SUSY case from Lecture 1).
    - Non-anomalous R-symmetry: DERIVE R[Q] = (N_f - N_c)/N_f from the requirement that the SU(N_c)^2 U(1)_R anomaly vanishes. Explicitly: Tr[T^a T^b R_fermion] = 0 requires N_f x T(fund) x (R[Q] - 1) + T(adj) x 1 = 0 (gaugino has R=1). This gives N_f (1/2)(R[Q] - 1) + N_c = 0, hence R[Q] = 1 - 2N_c/N_f = (N_f - N_c)/N_f - N_c/N_f... Actually: the condition is N_f T(fund) (R[Q]-1) + N_f T(fund) (R[Q-tilde]-1) + T(adj)(1) = 0. With R[Q] = R[Q-tilde] and T(fund) = 1/2: N_f (R[Q]-1) + N_c = 0, giving R[Q] = 1 - N_c/N_f = (N_f - N_c)/N_f. VERIFY: R[W] = 2 requires checking with the magnetic superpotential (done in Lecture 3).
    - State the table of quantum numbers (Table from RESEARCH.md: Electric Theory field content).

    Section 8: Moduli Space of SQCD (~5 min)
    - Flat directions: V_F = V_D = 0 parameterise the moduli space.
    - D-flat conditions for SU(N_c): |Q|^2 - |Q-tilde|^2 proportional to identity (D^a = 0).
    - Gauge-invariant coordinates: mesons M^i_j = Q^i Q-tilde_j (N_f x N_f matrix), baryons B^{i_1...i_{N_c}} = epsilon Q^{i_1}...Q^{i_N_c}, and B-tilde.
    - Classical constraint: rank M <= N_c; det M - B B-tilde = 0 for N_f >= N_c (classical).
    - Dimension counting: dim(moduli space) = 2 N_c N_f - (N_c^2 - 1) (matter fields minus gauge).
    - Preview: quantum corrections modify this constraint (Lecture 3).
