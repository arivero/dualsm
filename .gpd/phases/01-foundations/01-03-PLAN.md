    - Solution: Include the full calculation for each coefficient. SU(N_f)_L^3 = N_c (each of N_c colours contributes A(fund)=1). All coefficients expressed in terms of N_c, N_f.
    - Note in solution: these exact coefficients will be used to verify Seiberg duality in Exercise Set 3.

    **Problem 1: Beta function coefficient survey (warm-up, ~15 min)**

      - SU(3), N_f = 0 (pure Yang-Mills)

    **Problem 2: Banks-Zaks fixed point (~30 min)**
    - (a) Write the two-loop beta function: beta(alpha) = -(b_0/(2pi))alpha^2 - (b_1/(8pi^2))alpha^3.
    - (b) Show that a fixed point exists at alpha_* = -b_0/(2b_1) * (16 pi^2) when b_0 > 0 and b_1 < 0.
    - (c) For SU(3) with N_f = 16 (one flavour below AF boundary):
      Compute b_0 = 1/3, b_1 = (1/3)[34*9 - 16*(39 - 1)] = (1/3)[306 - 608] = (1/3)(-302) = -302/3.
      Compute alpha_* = (1/3)/((2)(302/3)) * 16pi^2 = (1/604) * 16pi^2 ~ 0.26.
    - (d) Is this perturbatively reliable? (alpha_* ~ 0.26 is borderline; state that for N_f closer to 16.5, alpha_* -> 0 and the perturbative expansion is controlled.)
    - (e) For SU(3) with N_f = 15: compute alpha_* and discuss reliability.
    - Solution: full numerical evaluation with discussion of perturbative reliability.

    **Problem 3: SUSY conformal window (~30 min)**
    - (a) State (without proof -- proof deferred to Lectures 2-3) that the exact NSVZ beta function for N=1 SQCD with SU(N_c) and N_f flavours gives the conformal window 3N_c/2 < N_f < 3N_c.
    - (b) Derive the upper boundary: at N_f = 3N_c, the theory becomes free (no interacting fixed point). Motivation: the NSVZ beta function numerator is proportional to 3N_c - N_f(1 - gamma_Q), and at N_f = 3N_c with gamma_Q = 0 (free theory), beta = 0.
    - (c) Derive the lower boundary: at N_f = 3N_c/2, the gauge-invariant operator M = QQ-tilde hits the unitarity bound dim(M) >= 1, which requires R[M] >= 2/3. Since R[M] = 2R[Q] = 2(N_f - N_c)/N_f, the bound R[M] >= 2/3 gives N_f >= 3N_c/2.
    - (d) Tabulate the SUSY conformal window for SU(2), SU(3), SU(5):
      - SU(2): 3 < N_f < 6, integers: N_f = 4, 5
      - SU(3): 4.5 < N_f < 9, integers: N_f = 5, 6, 7, 8
      - SU(5): 7.5 < N_f < 15, integers: N_f = 8, ..., 14
    - (e) Compare with the non-SUSY conformal window: state that the lower boundary is debated (N_f ~ 8-12 for SU(3)), in contrast with the exact SUSY result.
    - Solution: detailed derivation of each boundary with R-charge and unitarity bound.

    **Problem 4: Phases of gauge theories (~45 min, synthesis)**

      - N_f = 0: pure Yang-Mills (confinement, mass gap)
      - N_f = 1, ..., ~4: confinement with chiral symmetry breaking
      - N_f ~ 5-8: SUSY conformal window (IR fixed point)
      - N_f ~ 9-16: asymptotically free, Banks-Zaks regime (non-SUSY conformal window boundary debated)
      - N_f >= 17: loss of asymptotic freedom (free IR theory)
    - (b) For each regime, describe the IR physics (confined, conformal, free) and state what anomaly matching tells us about each.
    - (c) Explain why the conformal window is important for the dualised SM programme: if the SM gauge sectors can be dual to a UV theory in the conformal window, the duality map is under analytical control.
    - (d) Optional challenge: for SU(3) with N_f = 12 (near the lower boundary of the non-SUSY conformal window), discuss why lattice simulations are needed to settle the question and mention that lattice evidence is ambiguous (some studies find conformal, others find confinement).
    - Solution: include a TikZ figure showing the phase diagram.
