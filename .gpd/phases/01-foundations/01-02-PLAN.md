    - Use the \conventionbox macro.

    **Section 3: Quick review of gauge anomaly cancellation (~2 pages)**
    - Recall the ABJ anomaly (triangle diagram with three gauge boson insertions). Do NOT rederive from scratch -- Part III students have seen this.
    - State: for a gauge theory to be consistent, gauge anomalies must VANISH: Tr[T^a_gauge {T^b_gauge, T^c_gauge}] = 0 summed over all fermions.
    - SM example: show one generation of the SM cancels SU(3)^2 U(1)_Y and U(1)_Y^3 anomalies. State the result; do not re-derive.
    - Purpose: set up the contrast with 't Hooft anomaly matching.

    **Section 4: 't Hooft anomaly matching (~3-4 pages)**
    - CRITICAL: Present the explicit comparison table distinguishing gauge anomaly cancellation from 't Hooft anomaly matching.
      | Feature | Gauge anomaly | 't Hooft anomaly |
      | Symmetry type | Gauged | Global |
      | Condition | Must vanish | Must match (UV = IR) |
      | Physical meaning | Quantum consistency | Constraints on IR spectrum |
      | What happens if violated | Theory inconsistent | IR spectrum is wrong |
    - State the result: for every global symmetry with generators T^a, the anomaly coefficient A(a,b,c) = Tr[T^a {T^b, T^c}] must be equal when computed using UV degrees of freedom and IR degrees of freedom.
    - Present 't Hooft's spectator argument:
      (1) Promote global symmetry G_global to a weakly gauged symmetry (coupling g_weak -> 0).
      (2) Add spectator fermions to cancel the new gauge anomalies (this is always possible).
      (3) The combined theory (original + spectators) is consistent at all scales.
      (4) Spectators are free (g_weak -> 0), so their anomaly contribution is the same in UV and IR.
      (5) Therefore the anomaly from the original theory's fermions must also be the same in UV and IR.
    - Emphasise: anomaly matching is NECESSARY but NOT SUFFICIENT for determining the IR spectrum. It constrains but does not uniquely fix the confined spectrum.
    - State that this is exact by the Adler-Bardeen theorem: the anomaly is one-loop exact.

    **Section 5: Warm-up example: SU(3) QCD with N_f = 2 (~2-3 pages)**
    - UV spectrum: 2 Dirac quarks (u,d) in fund of SU(3)_c, with global SU(2)_L x SU(2)_R x U(1)_B.
    - Compute UV anomaly: SU(2)_L^3 anomaly = N_c * A(fund of SU(2)) = 3 * 1 = 3.
    - IR spectrum: 3 Goldstone bosons (pions) + proton + neutron. Baryons are in (2,1) of SU(2)_L x SU(2)_R (isospin doublet).
    - Compute IR anomaly: SU(2)_L^3 from baryons = 1 per baryon doublet. Need to match 3.
    - Discuss: the matching works with specific baryon representations. Present the argument that massless composite fermions must exist if anomaly matching is to be satisfied without massless scalars (i.e., without Goldstone bosons carrying the anomaly).
    - Note: this is a non-trivial constraint that rules out certain confinement scenarios.
    - Reference: 't Hooft 1980, Peskin TASI-96 hep-th/9702094.

    **Section 6: General anomaly matching for SU(N_c) with N_f flavours (~2 pages)**
    - List global symmetry: SU(N_f)_L x SU(N_f)_R x U(1)_B x U(1)_A (anomalous).
    - IMPORTANT: U(1)_A is anomalous (ABJ anomaly) and should NOT be matched. State this explicitly.
    - List independent anomaly coefficients to match:
      (1) SU(N_f)_L^3: A = N_c per fundamental Weyl fermion
      (2) SU(N_f)_R^3: A = N_c (by symmetry)
      (3) SU(N_f)_L^2 U(1)_B: depends on U(1)_B charge assignment
      (4) SU(N_f)_R^2 U(1)_B: by symmetry
      (5) U(1)_B^3
      (6) U(1)_B (gravitational anomaly if applicable)
    - Use B[Q] = 1/N_c, B[Q-tilde] = -1/N_c (Seiberg convention).
    - State: these constraints will be crucial for verifying Seiberg duality in Lecture 3.
    - Mark detailed anomaly coefficient computation as EXERCISE material (Exercise Set 1).

    **Section 7: Beta function and conformal window (~3-4 pages)**
    - State the one-loop beta function: mu d(alpha_s)/d(mu) = -b_0 alpha_s^2/(2pi) + ...
    - b_0 = (11N_c - 2N_f)/3 for N_f Dirac flavours.
    - Review the derivation logic (do not rederive): gluon self-coupling gives +11N_c/3, fermion loops give -2N_f/3.
    - Loss of asymptotic freedom: b_0 = 0 at N_f = 11N_c/2. For SU(3): N_f = 16.5, so N_f >= 17 loses AF.
    - Two-loop beta function: state b_1 = (1/3)[34 N_c^2 - N_f(13 N_c - 3/N_c)].
    - Banks-Zaks fixed point: when b_0 > 0 but b_1 < 0, beta function has a zero at alpha_* = -(b_0/b_1) * 8pi^2. The theory flows to a non-trivial IR fixed point.
    - Conformal window: the range of N_f where the theory is asymptotically free but flows to an IR fixed point rather than confining.
    - Non-SUSY: upper boundary clear (N_f = 11N_c/2); lower boundary debated (N_f ~ 8-12 for SU(3)). State 2-loop estimate is APPROXIMATE.
    - SUSY conformal window (preview): 3N_c/2 < N_f < 3N_c is EXACT (follows from the NSVZ beta function, to be introduced in Lecture 2-3).
    - For SU(3): SUSY window is 4.5 < N_f < 9, i.e., N_f = 5,6,7,8.
    - R-charge at fixed point: R[Q] = (N_f - N_c)/N_f. At N_f = 3N_c: R[Q] = 2/3 (unitarity bound saturated; theory becomes free). At N_f = 3N_c/2: confinement sets in.
    - Mark detailed conformal window calculations as EXERCISE material (Exercise Set 2).
