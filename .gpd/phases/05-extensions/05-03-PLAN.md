    **Exercise 7.1: SO(N_c) Anomaly Matching (30 marks)**

    Problem: Consider SO(10) with N_f = 7 vector representations Q_i (i = 1,...,7).
    (a) [5 marks] State the dual theory: gauge group, matter content, superpotential. What is the conformal window for SO(10)?
    (b) [10 marks] Compute the SU(7)^3 and SU(7)^2 U(1)_R anomaly coefficients on the electric side.
       - Electric: 10 copies of the vector representation of SU(7). Each contributes A(fund SU(7)) to the cubic anomaly. So SU(7)^3_el = 10.
       - For SU(7)^2 U(1)_R: need R-charges. R[Q_el] follows from anomaly-free U(1)_R: R[Q] = 1 - N_c/N_f (same logic as SU case but with h = N_c - 2). Actually, for SO(N_c) with N_f vectors, R[Q] = 1 - (N_c - 2)/N_f = (N_f - N_c + 2)/N_f.
    (c) [10 marks] Compute the same anomaly coefficients on the magnetic side: SO(7 - 10 + 4) = SO(1). But SO(1) is trivial! This means N_f = 7 is below the conformal window for SO(10): the theory s-confines. Discuss what this means.
       - Actually check: conformal window lower bound = 3(N_c-2)/2 = 3*8/2 = 12. So N_f = 7 < 12. This is in the confined phase, not the conformal window. The theory either s-confines or is in a different phase.
       - CORRECTION: Use SO(10) with N_f = 13 instead (inside conformal window: 12 < 13 < 24). Magnetic: SO(13 - 10 + 4) = SO(7) with 13 vectors + symmetric meson. Compute anomaly coefficients for this case.
    (d) [5 marks] Verify the SU(13)^3 anomaly coefficient matches between electric SO(10) and magnetic SO(7) theories.

    SOLUTION: Full worked solution with explicit computation of each anomaly coefficient, using T(vector SO(N)) = 1 convention. Show the meson contribution to the magnetic-side anomaly (the symmetric tensor representation of SU(N_f) contributes to the cubic anomaly).

    **Exercise 7.2: Kutasov-Schwimmer Duality (20 marks)**

    Problem: Consider SU(4) with N_f = 3 fundamentals Q_i, Q-tilde^i and one adjoint Phi with superpotential W = g Tr(Phi^3)/3 (i.e., k = 2).
    (a) [5 marks] State the dual gauge group and matter content. How many meson species M_j are there?
       - Dual: SU(kN_f - N_c) = SU(2*3 - 4) = SU(2) with 3 duals q, q-tilde, adjoint phi, and mesons M_0, M_1 (j = 0, 1 since k = 2, so j = 0,...,k-1 = 0,1).
    (b) [5 marks] Write the magnetic superpotential explicitly for this case.
    (c) [5 marks] Show that for k = 1, the KS duality reduces to Seiberg duality: the adjoint gets a mass term W = Tr(Phi^2)/2, integrate it out, and recover the standard Seiberg dual.
    (d) [5 marks] What is the conformal window for this theory? Verify that N_f = 3 is inside it.
       - Conformal window: 3N_c/(k+1) < N_f < 3N_c/k = 3*4/(2+1) < N_f < 3*4/2 -> 4 < N_f < 6. But N_f = 3 < 4, so it is BELOW the conformal window. CORRECTION: Use SU(6) with k=2, N_f=4. Then 3*6/3 < 4 < 3*6/2 -> 6 < 4? No. Try SU(3) with k=2, N_f=2. Conformal window: 3 < N_f < 4.5. N_f=2 is below. Use SU(3) with k=2, N_f=4. Then 3 < 4 < 4.5. Yes, inside. Dual: SU(2*4-3) = SU(5). Hmm, that's a higher rank. Try SU(5) with k=2, N_f=4. Window: 5 < 4? No. This is getting complicated.
       - SIMPLIFY: Use the k=1 limit problem as part (c) only. For part (d), ask students to derive the conformal window formula 3N_c/(k+1) < N_f < 3N_c/k and verify it reduces to 3N_c/2 < N_f < 3N_c for k=1.

    SOLUTION: Full worked solution.

    (c) [5 marks] Verify that this is the boundary of Seiberg duality: the magnetic gauge group SU(N_f - N_c) = SU(1) is trivial. What does this tell you about the connection between s-confinement and duality?

    **Exercise 8.1: Meson Quantum Numbers (25 marks)**

    Problem: Consider the 3x3 meson matrix for the light quarks (u, d, s).

    (b) [10 marks] For each off-diagonal meson, compute:
       - Isospin I and I_3
       - Strangeness S
       - Electric charge Q from the Gell-Mann-Nishijima formula Q = I_3 + (B + S)/2 (with B = 0 for mesons)
       - Verify Q matches the known charge of each meson
    (c) [5 marks] The diagonal entries M_{11}, M_{22}, M_{33} correspond to flavour-neutral combinations. Explain why the physical states are not simply u u-bar, d d-bar, s s-bar but rather linear combinations (pi^0, eta, eta'). What symmetry principle determines the mixing?
    (d) [5 marks] Extend to N_f = 4 by including the charm quark. Identify the new entries M_{14} = u c-bar, M_{24} = d c-bar, M_{34} = s c-bar with physical mesons and state their masses (PDG values).

    SOLUTION: Full worked solution.

    - (b) Explicit quantum number computation for each meson:

      K^+: u s-bar, I=1/2, I_3=+1/2, S=+1, Q = 1/2 + 1/2 = +1. Check: Q(u) - Q(s) = 2/3 - (-1/3) = +1.
      K^0: d s-bar, I=1/2, I_3=-1/2, S=+1, Q = -1/2 + 1/2 = 0. Check: Q(d) - Q(s) = -1/3 - (-1/3) = 0.
      And conjugates.
    - (c) The physical neutral mesons are determined by the flavour SU(3) representation decomposition: 3 x 3-bar = 8 + 1. The octet gives pi^0, eta; the singlet gives eta'. The mixing is determined by SU(3) flavour symmetry (broken by m_s != m_u, m_d).
    - (d) D^0 = c u-bar (1865 MeV), D^+ = c d-bar (1870 MeV), D_s^+ = c s-bar (1968 MeV).

    **Exercise 8.2: Scalar vs Pseudoscalar Mesons (20 marks)**

    Problem: In the SUSY dual theory, the meson M_{ij} is a chiral superfield with a scalar lowest component (J^P = 0^+). Physical pions are pseudoscalars (J^P = 0^-).
    (a) [5 marks] Write the decomposition of the meson field around its VEV in the linear sigma model:
       M_{ij} = (v_{ij} + sigma_{ij} + i pi_{ij}) / sqrt(2)
       Identify sigma as the scalar mode (J^P = 0^+) and pi as the pseudoscalar mode (J^P = 0^-). What physical particles does each correspond to?
    (b) [10 marks] Show that the pseudoscalar mesons are Goldstone bosons of chiral symmetry breaking SU(N_f)_L x SU(N_f)_R -> SU(N_f)_V:
       - How many Goldstone bosons are there for N_f = 2? For N_f = 3?
       - Why are pions massive if they are Goldstone bosons? (pseudo-Goldstones from explicit chiral symmetry breaking by quark masses)
       - Use the GMOR relation m_pi^2 f_pi^2 = -(m_u + m_d) <q-bar q> with f_pi = 92 MeV, m_u + m_d = 8.5 MeV, <q-bar q> = -(270 MeV)^3 to compute m_pi. Compare with the observed 140 MeV.
    (c) [5 marks] Why is this distinction important for the dualised Standard Model? The SUSY meson M_{ij} maps to the scalar (sigma-like) mesons after SUSY breaking, while the pseudoscalar pions emerge independently from chiral symmetry breaking. Explain why the statement "the pion IS the dual meson" is incorrect without qualification.

    SOLUTION: Full worked solution.
    - (a) sigma -> f_0(500)/sigma, a_0(980), K_0*(700); pi -> pi(140), K(494), eta(548), eta'(958)
    - (b) N_f=2: 3 Goldstones (pi triplet); N_f=3: 8 Goldstones (pseudoscalar octet)
      GMOR: m_pi^2 = (8.5 MeV)(270 MeV)^3/(92 MeV)^2 = 8.5 * 1.97e7 / 8464 MeV^2 = 19780 MeV^2 -> m_pi = 141 MeV. Matches!
    - (c) The SUSY meson superfield has a scalar lowest component. The pion is a pseudoscalar arising from the phase fluctuation of the chiral condensate. These are distinct physical states.

    **Exercise 8.3: Mass Hierarchy from VEV Structure (20 marks)**

    Problem: In the dualised SM (Lecture 8), the effective Yukawa couplings are Y_{ij} = y <H_{ij}>/v with a single universal coupling y.
    (a) [5 marks] The VEV hierarchy has three tiers: <H_{33}> >> <H_{22}> >> <H_{11}>. If <H_{33}^u> ~ v (the dominant VEV), what is the top Yukawa y_t? What does this imply for m_t?
    (b) [10 marks] Given the observed quark mass ratios:
       m_t/m_c ~ 130, m_c/m_u ~ 550, m_b/m_s ~ 50, m_s/m_d ~ 20
       Show that a geometric hierarchy <H_{(n+1)(n+1)}>/<H_{nn}> ~ epsilon with epsilon ~ 0.05-0.2 (depending on the sector) can reproduce these ratios with O(1) coefficients. (This is a parametric consistency check, NOT a prediction.)
    (c) [5 marks] Relate the meson mass hierarchy to the VEV hierarchy. For the pseudoscalar mesons in the light sector, the meson masses are determined by QCD (quark masses + chiral dynamics), not directly by the VEVs. Explain why the dualised SM framework is consistent with, but does not improve upon, standard QCD predictions for meson masses. What DOES the dualised SM add to the picture?

    SOLUTION: Full worked solution.
    - (a) y_t = y <H_{33}^u>/v ~ y ~ O(1). So m_t = y_t v ~ 174 GeV (parametric consistency). Note: y is NOT predicted to be exactly 1; it is O(1) by assumption.
    - (b) If <H_{33}>/<H_{22}> ~ epsilon and <H_{22}>/<H_{11}> ~ epsilon, then:
      m_c/m_t ~ epsilon, m_u/m_c ~ epsilon -> m_u/m_t ~ epsilon^2
      For epsilon ~ 1/130^{1/1} ~ 0.008 for top-charm -> this gives too small values for up-charm.
      Actually the VEV hierarchy is not a simple geometric series -- the ratios are different for u-type and d-type, and involve different O(1) coefficients c^{abcd}. The solution should show that WITH free O(1) coefficients, any set of mass ratios can be accommodated. This is parametric consistency, not a prediction.
    - (c) The dualised SM adds: (1) a UV origin for the meson matrix structure, (2) the universal Yukawa coupling y, (3) a framework for the mass hierarchy (Planck-scale operators with O(1) coefficients). It does NOT predict individual meson masses.
